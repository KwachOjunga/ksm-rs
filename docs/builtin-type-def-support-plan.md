# Builtin Type Definition Support Plan

This is related to [struct declaration](./struct-declaration-plan.md).
The defining of struct types has neccessesitated the introduction of field 
names to allow for struct field access and guide how the struct 
layout is defined.

Kisumu lang will support various builtin type definitions;

- integers
  - i32,i64,u32,u64,
- floats
  - f32,f64
- strings
- boolean
- void
